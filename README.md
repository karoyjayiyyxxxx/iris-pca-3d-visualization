# Iris Dataset PCA 3D Visualization

This project demonstrates how to perform **Principal Component Analysis (PCA)** on the classic Iris dataset using `scikit-learn` and visualize the results in a 3D scatter plot using `matplotlib`.

The Iris dataset consists of 4 features (sepal length, sepal width, petal length, petal width). Humans cannot visualize 4-dimensional data directly. 

This script:
1. Loads the Iris dataset.
2. Uses PCA (Principal Component Analysis) to reduce the dimensionality from 4 dimensions to 3 dimensions.
3. Visualizes the transformed data in a 3D space, showing how well the three classes (Setosa, Versicolour, Virginica) are separated.
